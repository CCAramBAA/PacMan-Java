## ✨ 功能特性

### 🎯 核心玩法
- ✅ **经典迷宫布局**：完美还原 28×31 的经典地图设计
- ✅ **流畅的移动系统**：平滑的吃豆人移动与转向机制
- ✅ **完整的豆子系统**：240 个普通豆子 + 4 个能量豆
- ✅ **穿墙隧道功能**：左右隧道传送，策略性逃生
- ✅ **精确碰撞检测**：像素级碰撞判定

### 👻 智能幽灵 AI 系统

每个幽灵都有独特的性格和追击策略，完全还原原版游戏：

| 幽灵 | 颜色 | AI 行为 | 特点 |
|------|------|---------|------|
| **Blinky** 🔴 | 红色 | 直接追击 | "影子"，始终追踪吃豆人当前位置 |
| **Pinky** 🩷 | 粉色 | 预判伏击 | "速度型"，预判吃豆人前方 4 格位置 |
| **Inky** 🩵 | 青色 | 协同包围 | "害羞的"，基于 Blinky 位置进行向量计算 |
| **Clyde** 🧡 | 橙色 | 忽远忽近 | "笨拙的"，距离远时追击，近时逃跑 |

#### 🎭 四种幽灵状态
- **追击模式 (Chase)** 🔥：主动追捕吃豆人，持续 20 秒
- **散射模式 (Scatter)** 💨：返回各自角落，持续 7 秒
- **恐惧模式 (Frightened)** 😱：深蓝色，随机移动，可被吃掉
- **眼睛模式 (Eaten)** 👀：只剩眼睛，快速返回幽灵屋重生

### 🎮 丰富的游戏系统

#### 📊 计分系统
- 普通豆子：**10 分**
- 能量豆：**50 分**
- 吃幽灵：**200 → 400 → 800 → 1600**（连续翻倍）
- 水果奖励：**100 ~ 5000 分**（根据关卡变化）
- 额外生命：**10000 分**自动奖励
- 最高分自动保存至本地文件

## 🚀 快速开始

### 环境要求
- **JDK 17** 或更高版本
- **IntelliJ IDEA**（推荐）或其他 Java IDE
- 支持 Java Sound API 的操作系统

### 安装步骤

#### 方法 1：使用 IntelliJ IDEA（推荐）

1. **克隆仓库**
2. **打开项目**
   - 启动 IntelliJ IDEA
   - 选择 `File` → `Open`
   - 选择项目文件夹
   - IDEA 会自动识别项目结构

3. **运行游戏**
   - 找到 `src/game/Game.java`
   - 右键点击 → `Run 'Game.main()'`
   - 或使用快捷键 `Shift + F10`

#### 方法 2：命令行编译运行
#### 方法 3：使用 Maven/Gradle（可选）

如需构建工具支持，可自行添加 `pom.xml` 或 `build.gradle` 配置文件。
